# PHP Algo - Exercices

## Exercice 1 ##

Donner les valeurs de $x, $y, $z à la fin du script suivant :
```
$x = "PostgreSQL"; 
$y = "MySQL"; 
$z = &$x; 
$x = "PHP 5"; 
$y = &$x;
```
